# 更新日誌 (Changelog)

所有關於本專案的重大變更都將紀錄在此檔案中。

## [Web Edition 1.0.2] - 2026-03-31

### 🚀 新增 (Added)
- **雲端模擬測試模式 (Simulation Mode)**：支援輸入 `TEST` 與 `0000` 進入模擬環境，並提供預設的範例存檔供測試。
- **故障找回機制 (Code Recovery)**：自動在瀏覽器 `localStorage` 備份最新的轉移碼與認證碼，並於後端日誌同步印出，防止因網頁重新整理導致代碼遺失。
- **在地化回饋 (Localization)**：將後端所有的上傳、登入、修改錯誤提示全面「繁體中文化」。

### 🔧 修復與優化 (Fixed & Optimized)
- **安全數值限制 (Item Limits)**：強制執行各項道具的合理上限（如貓罐頭 45,000），防止因數值異常導致帳號封鎖。
- **道具名稱一致性**：修正了「貓薄荷種子」命名不一致與「迷宮獎牌」名稱缺失問題。

## [Web Edition 1.0.0] - 2026-03-13

### 🚀 新增 (Added)
- **全新 Web 介面**: 將 BCSFE-Python 核心轉換為視覺化網頁操作介面。
- **進階一鍵優化套餐**: 支援全貓獲得、全科技升級、主線全金寶一鍵完成。
- **自動備份系統**: 每次登入時自動備份原始存檔，確保安全。
- **Web 伺服器支援**: 基於 FastAPI 開發，支援穩定、高效的後端處理。

### 🔧 修復與優化 (Fixed & Optimized)
- **存檔讀取修復**: 解決了 `BattleItem` 屬性名稱衝突導致的 500 錯誤。
- **存檔上傳修復**: 修正了迷宮獎牌 (Labyrinth Medals) 與幸運券資料類型的 422 驗證錯誤。
- **Session 管理**: 修復了因伺服器重啟導致的 Session 遺失問題。
- **在地化更新**: 提供完整的繁體中文 README 與更新日誌。

---

## [3.2.2] - 2025-12-29

### 新增 (Added)
- 貓神社 (Cat Shrine) 功能選單：可控制其在遊戲中顯示或隱藏。

### 變更 (Changed)
- 重新設計了非故事章節的關卡清除邏輯：現在地圖進度與清除次數會分開設定。

### 修復 (Fixed)
- 修復了遊戲數據下載的多個問題。
- 修正了「停用最大值」功能在某些項目下失效的問題。
- 改進了獲取地圖名稱的速度。
- 修復了當停用最大值並輸入過大升級值時，特殊技能升級導致程式崩潰的問題。

## [3.0.0] - 2025-08-04

- 新增寶箱 (Treasure Chests) 修改。
- 支援全新的本能珠系統。
- 支援超進化 (Ultra Form) 修改。
- 改進存檔備份機制。
- 支援迷宮獎牌 (Labyrinth Medals) 修改。
