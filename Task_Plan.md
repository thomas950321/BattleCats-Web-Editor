## 開發守則

1. **每個 Ticket 完成後**，必須同步更新 `Dev_Status.md`
2. **每修改 `service.py`**，必須確認 `models.py` 型別定義一致
3. **所有 UI 改動**，必須本地 `python bcsfe_web/main.py` 確認後才 Push
4. **涉及數值修改的功能**，Clamp 邏輯不可低於既有安全標準（catfood ≤ 45,000）
