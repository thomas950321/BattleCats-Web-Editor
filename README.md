# 貓咪大戰爭修改器 (BattleCats-Web-Editor) - 繁體中文版

BattleCats-Web-Editor 是一個專為《貓咪大戰爭》開發的網頁版修改器。

---

## 💡 專案致謝與起源

本網頁版修改器是基於 **[fieryhenry/BCSFE-Python](https://codeberg.org/fieryhenry/BCSFE-Python)** 核心引擎進行的深度優化與 Web 化改造。

### 特別感謝：
*   **fieryhenry**: 提供強大的 `BCSFE-Python` 核心邏輯。
*   **Lethal**: 提供最初的存檔修改與 CF/XP 修正啟發。

---

## 💻 安裝說明

### Windows / MacOS
1.  安裝 Python 3.9 或以上版本。
2.  開啟 PowerShell 或終端機，執行：
    ```powershell
    pip install bcsfe
    ```
3.  啟動修改器：
    ```powershell
    bcsfe
    ```

### Android (使用 Termux)
1.  安裝 Termux。
2.  執行以下指令：
    ```sh
    pkg install python python-pip
    pip install bcsfe
    ```
3.  執行 `bcsfe` 即可開始。

## ☁️ 雲端部署 (Render)

本專案支援一鍵部署至 **Render.com**。

### 部署步驟：
1.  **Fork 本專案**：將專案 Fork 到您自己的 GitHub 帳號。
2.  **連結 Render**：在 Render 儀表板選擇「Blueprint」，連結您的 GitHub 倉庫。
3.  **自動化配置**：Render 會自動讀取 `render.yaml` 並完成所有環境設定。
4.  **開始使用**：部署完成後，您將獲得一個專屬的網址（例如 `https://your-app.onrender.com`）。

> [!NOTE]
> 使用免費版 Render 時，伺服器在一段時間沒人存取後會進入休眠。重新喚醒約需 30 秒。

## 🛠️ 使用方法 (引繼流程)

1.  在遊戲內「設定」->「更換帳號/設備」點擊「**開始資料轉移**」。
2.  點擊「**儲存資料至伺服器**」，取得並記下「**引繼碼**」與「**認證碼**」。
3.  **使用 Web 介面 (推薦)**：
    *   啟動網頁伺服器後，在瀏覽器輸入您的代碼。
    *   修改完成後點擊「**儲存並上傳至伺服器**」。
4.  **回遊戲回復資料**：
    *   點擊「取消資料轉移」（時間過久需重啟遊戲）。
    *   在「更換帳號/設備」選單中，點擊「**繼續資料轉移**」並輸入**新代碼**。

## ⚠️ 使用條款與安全建議

*   **貓罐頭**：強烈建議保持在 **45,000** 以下，以免被系統偵測鎖號。
